# O que é?
Pacote para arrumar as chaves de registro e políticas para criar uma rede entre versões diferentes de Windows (com versões extremamente antigas).

# Como utilizar?
1. Executar o arquivo .bat apenas nas unidades aonde é desejado criar a pasta Laboratório na unidade Y:;
2. Mesclar todos os arquivos .reg no computador de origem;
3. Reiniciar o computador origem.
