; 記述方法
; ・0カラム目から "[" + 4桁の16進数値 + "]" の指定で、指定番号のコンテンツ
;   データ開始を示します。
; ・文章中に "{" + BGR 16進数コード6桁 + "C}" で囲んだ文字列があると、RGB 
;   の色コードとしてその後の文字の色を変更します。
;   ただし、改行を行うと、黒に戻ります。
; ・文章中に "{" + BGR 16進数コード6桁 + "B}" で囲んだ文字列があると、RGB 
;   の色コードとしてその後の文字の背景色を変更します。
;   背景色に、FFFFFF を与えるとデフォルト色に戻します。
;   また、改行を行うと、デフォルトの色に戻ります。
